🔧 Initialiser le Schéma Backup

Créer les tables de sauvegarde cloud dans Supabase

1 Ouvrir le Dashboard Supabase
Rendez-vous sur app.supabase.com et connectez-vous.
2 Aller dans SQL Editor
Sélectionnez votre projet CharisHub, puis cliquez sur "SQL Editor" dans le menu de gauche.
3 Copier le SQL
Ouvrez le fichier supabase-backup-sync.sql (dans le dossier du projet) et copiez TOUT son contenu.
4 Exécuter le Script
Collez le SQL dans l'éditeur, puis cliquez sur "Run" (ou Ctrl+Enter).
💡 Astuce : Après avoir créé les tables, rechargez CharisHub et vous verrez le badge "Connecté" vert en bas à droite.
